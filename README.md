# el
**Extension lookup**

* Supports 1,646 file extensions
* Extensible

## Quick Start
* Clone the repo: `git clone https://github.com/pepa65/el`
* Compile: `make`
* Move `el` into the PATH: `mv el ~/bin/`

## Usage
`el` <filename|extension>

### Examples
`el crt`
```
.crt: Distinguished Encoding Rules stores certificates
```

`el el.c`
```
.c: C language file
```
